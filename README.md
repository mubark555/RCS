# إدراج الأسماء في PowerPoint من Excel (تطبيق ويب)

تطبيق ويب بسيط: ترفع **قالب PowerPoint** فيه مكان محجوز للاسم، و**ملف Excel** فيه قائمة الأسماء،
فيُنشئ لك ملف PowerPoint فيه **شريحة لكل اسم** (أو ملف مضغوط فيه ملف مستقل لكل اسم).
مبني بـ **Next.js** وكل المعالجة تتم **داخل المتصفح** — لا تُرفع ملفاتك لأي خادم — لذا يعمل مباشرة على **Vercel**.

## التشغيل محليًا

```bash
npm install
npm run dev
```

ثم افتح <http://localhost:3000>.

## النشر على Vercel

1. ارفع هذا المستودع إلى GitHub (تم بالفعل على فرع التطوير).
2. من [vercel.com](https://vercel.com) اختر **New Project** ثم اربطه بالمستودع.
3. Vercel سيتعرّف تلقائيًا أنه مشروع **Next.js** — اضغط **Deploy** بدون أي إعدادات إضافية.

> لا يحتاج أي متغيّرات بيئة ولا قاعدة بيانات؛ كل شيء يعمل في المتصفح.

## طريقة الاستخدام في 3 خطوات

1. جهّز قالب PowerPoint وضع في مكان الاسم النص المحجوز: `{{name}}`.
2. جهّز ملف Excel واكتب الأسماء في عمود واحد.
3. ارفع الملفين في الصفحة، اختر طريقة الإخراج، واضغط **توليد الملف**.

### الخيارات في الواجهة

| الخيار | الوصف | الافتراضي |
| --- | --- | --- |
| النص المحجوز | النص الذي يُستبدل بالاسم داخل الشريحة | `{{name}}` |
| عمود الأسماء | حرف مثل `A` أو رقم مثل `1` | أول عمود |
| اسم الورقة | اسم الورقة داخل ملف Excel | أول ورقة |
| الصف الأول عنوان | تجاهُل الصف الأول (عنوان العمود) | مُفعّل |
| طريقة الإخراج | ملف واحد (شريحة لكل اسم) أو ملف لكل اسم داخل zip | ملف واحد |

## كيف يعمل؟

- ملف `.pptx` هو في الأصل ملف مضغوط يحوي XML؛ نفتحه في المتصفح عبر [JSZip](https://stuk.github.io/jszip/).
- نقرأ الأسماء من Excel عبر [SheetJS](https://sheetjs.com/).
- ننسخ شريحة القالب لكل اسم ونستبدل المكان المحجوز مع **الحفاظ على تنسيق النص** (الخط، الحجم، اللون)،
  ويشمل الاستبدال مربّعات النص والجداول.
- نعيد ضغط الملف ونُنزّله مباشرة.

## بنية المشروع

```
app/               صفحات Next.js والواجهة
  page.js          الواجهة الرئيسية
  layout.js        الهيكل العام (RTL عربي)
  globals.css      التنسيق
lib/
  pptx.js          توليد PowerPoint (نسخ الشرائح والاستبدال)
  excel.js         قراءة الأسماء من Excel
python-cli/        نسخة سطر أوامر بلغة بايثون (اختيارية، بديلة عن الويب)
```

## نسخة سطر الأوامر (اختيارية)

يوجد داخل `python-cli/` نفس الأداة كسكربت بايثون لمن يفضّل التشغيل من الطرفية:

```bash
cd python-cli
pip install -r requirements.txt
python make_samples.py    # ملفات تجريبية
python name_inserter.py --pptx template.pptx --excel names.xlsx --output result.pptx --has-header
```
